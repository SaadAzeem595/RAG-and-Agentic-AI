# 🤖 IBM RAG & Agentic AI Practice

A hands-on repository dedicated to mastering **Retrieval-Augmented Generation (RAG)** and **Agentic AI workflows** through practical labs, implementations, and mini-projects.

This repository showcases my learning journey across modern LLM orchestration frameworks such as **LangChain**, **LangGraph**, **CrewAI**, **AG2/AutoGen**, and **IBM Watsonx.ai**, with a strong focus on building **intelligent, tool-using, retrieval-enhanced AI systems**.

## 🧠 Highlighted Projects

### 🔁 Reflection Agent (LangGraph)

Built a **self-correcting agent loop** where:
- an LLM generates an initial response,
- a **critic/reflection agent** evaluates it,
- and the response is refined iteratively for better quality.

---

### 🏥 Healthcare Multi-Agent Chatbot

Developed a **multi-agent healthcare assistant** using **AG2 / AutoGen**, where different agents specialize in specific medical or support roles to improve response quality and domain relevance.

> **Note:** This project is for educational and experimentation purposes only.

---

### 🥗 Structured Meal & Grocery Planner

Created a **CrewAI-powered multi-agent system** for automated meal planning.

#### Agent Roles:
- **Nutritionist Agent** → creates meal recommendations
- **Grocery Specialist Agent** → generates shopping lists and planning support

This project demonstrates **task decomposition**, **role specialization**, and **agent collaboration**.
